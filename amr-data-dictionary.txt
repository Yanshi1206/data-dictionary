AMR data

Last updated on 2026/09/18

Author: Yanshi
Editor: Ella 
100,000 rows of 12 variables
These data represent the sort of data that might be obtained from the 2nd generation surveillance system.

* id Integer - A unique identifier for each person
* dob Date - Date of birth for each person
* spec_date Character - a string giving the date a specimen was taken
* organism Character - indicates the specimen name
